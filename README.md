## 💻 环境及需要的账户

- 安装Python虚拟环境
- 能注册DAWN的邮箱号
- 代理IP（可选）

---

## 🛠️ 设置

1. 克隆仓库：
   ```bash
   git clone https://github.com/GzGod/Dawn_ProMax
   ```
2. 创建并激活虚拟环境：
   ```bash
   Windows系统：
   python -m venv venv
   cd venv/Scripts
   activate
   Linux（服务器）：
   python3 -m venv venv
   cd venv/bin
   source activate
   cd ../..
   ```
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```


- **Email Verification Issues**：检查`settings.yaml`中的电子邮件提供商IMAP设置。
